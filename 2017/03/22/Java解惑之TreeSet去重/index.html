  • TreeMap内置默认的Comparator比较器，一旦设定，后续所有的比较必须依据此比较器进行;发现不符合比较器方式的，抛出异常。
  • 核心的两个do-while循环，依据比较器结果变量cmp，判断是将值插入的觉提位置。一旦cmp=0将value放到相同位置。
